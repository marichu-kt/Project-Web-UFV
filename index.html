What is HTML?

HTML, which stands for "Hypertext Markup Language", is the standard used for creating and designing web pages. It was created by the World Wide Web Consortium (W3C) and has become the basis for most web content. HTML uses a markup structure composed of labels or tags that surround and describe the content of the page. These tags tell the web browser how to display and organize elements on the screen. HTML is used in web development and is the cornerstone for creating web pages.

What is CSS?

CSS, which stands for "Cascading Style Sheets" in English, is a styling language used to describe the presentation of a document written in HTML or XML, including aspects such as layout, colors and fonts. Instead of including these styles directly in the HTML code, CSS allows you to separate the structure of the document (HTML) from its presentation (CSS), making it easier to manage and maintain the website.

What is JS?

JavaScript is a high-level, interpreted, object-oriented programming language used primarily for the development of interactive web pages. It was originally created by Netscape and has become one of the most popular and widely used programming languages in web development. JavaScript is used in a variety of contexts and environments due to its versatility. It is characterized by several distinctive features that make it unique and versatile.

What is a LICENCE file?

A license file (in Spanish,licenses) is a legal document that establishes the terms and conditions under which the right to use software, artistic work, or any other type of intellectual property is granted. It is essentially a contract between the rights holder (licensor) and the person or entity obtaining the license (licensee). The main purpose of a license is to define the limits and restrictions on how the copyrighted work can be used, distributed, modified, and shared.

What is a README file?

A README file, which stands for "Read Me" in English, is a document that is usually included in a software project to provide essential information about it. Its main purpose is to guide users, collaborators and developers on how to use, contribute to or understand the project. The README file, an abbreviation for "Read Me," is a critical component in any software project. This document is intended to provide essential information about the project, guide users and developers, and facilitate understanding and collaboration in the development and use of the software.
