Proxychains in Linux is another tool for anonymity providing anonymity and safe browsing with proxychains is easy. The proxychains works on socks4, socks5, HTTP, and https protocols. Setting up proxychains is easy but many users get errors while using them, some of the most common errors occur during tor installation and other errors like proxychain starts but you are not anonymous and your DNS leaks appear.

For more information, I leave a video tutorial explaining what ProxyChains is and its differences compared to a normal proxy, how to use it and why, this video belongs to the HackerSploit channel, I recommend that you go watch it because it has very interesting things that may be of interest .

First, update the Linux system with the patches and the latest applications. For this we open a terminal and type:


Terminal
$ sudo apt-get update && sudo apt-get upgrade

After the update and update we need to install tor, we will do it with the following command:


Terminal
$ sudo apt-get install proxychains tor -y

We had already installed tor service before so there will be a difference in execution. If you get any error while doing this then check below steps in this article. After installing tor service, now we need to configure proxychains. To do so use the following command.


Terminal
$ sudo nano /etc/proxychains4.conf

You can use any editor of your choice here we are using nano, you can also use vim, leafpad or sublime etc.

In the configuration, you will see “#” which means bash language comments. You can use arrow keys to scroll down and do the following changes.

- Remove Dynamic chain from comment

- Comment Strict chain and Random chain

- Remove proxy DNS from comment

- Write http/socks4/socks5 127.0.0.1 9050 in last line

This are the changes that we have changed, leave it like this EXAMPLE, just follow the instructions.


Terminal

dynamic_chain                                    

#strict_chain                                    

proxy_dns                                        

tcp_read_time_out 15000 tcp_connect_time_out 8000

http 127.0.0.1 9050                             

socks4 127.0.0.1 9050                            

socks5 127.0.0.1 9050                            

Save the configuration file and exit the terminal. The proxychains setup is complete. To start proxychains first restart the tor service and then launch proxychains in firefox with a link for a particular search engine like bing, duckduck go. Use the following commands:


Terminal

service tor restart                   

proxychains firefox www.duckduckgo.com

Good news, you are already under a chain of Tor proxies, which will allow you to browse the Internet in a more secure way, your IP will constantly change so it will be more difficult to track you, stay safe = be responsible.
